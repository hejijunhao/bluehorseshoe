Idea for a simple tool/platform:

- A list of Stocks/public equities are added to a watchlist/library
- Each has a profile page (looks like Perplexity's stock overview page/profile - will share later)
- A digest/watch monitor that monitors each stock perpetually perhaps e.g. every 6, 12, 24 or 48h (just as an example)
- The monitoring focuses on key things across socials (e.g. posts on X) as well as things like financial reporting etc. etc.

The use case is for me to be able to add stocks I hold and/or am thinking about investing in and want to monitor 24/7.

A Hermes Agent (from Nous Research) could be deployed to run this, but this too we will get into later, as it's more a mechanism than the foundational tool in and of itself.
